# SCAME 选型工具 1.0 - 开发说明与使用指南

## 概述

**SCAME 产品智能选型工具 1.0** 是一个基于 React + TypeScript + RAG (检索增强生成) 技术的工业插头插座智能选型系统。本工具严格遵循 SCAME 官方技术规范，为企业用户提供准确、高效的工业电气产品选型解决方案。

### 核心价值
- **技术准确性**: 严格遵循 SCAME 编码规则，杜绝技术参数幻觉
- **智能推荐**: 基于 SCAME "黄金替换法则" 的智能产品推荐
- **知识增强**: RAG 系统集成官方手册和技术文档
- **用户体验**: 直观的界面设计和流畅的选型流程

### 版本信息
- **版本号**: 1.0.0
- **发布日期**: 2026-04-15
- **开发团队**: 北京韶聪泽明智能科技有限责任公司
- **技术栈**: React 18 + TypeScript + Vite + Tailwind CSS

**路径**: `/`
**功能**: 系统概览和快速入口
- 欢迎界面和工具简介
- 主要功能模块导航
- 快捷选型入口
- 系统状态显示

### 2. 参数化选型 (ForwardSelectionPage)
**路径**: `/forward-selection`
**功能**: 基于技术参数的正向选型
- **电流规格选择**: 16A - 800A (包含标准电流和大电流)
- **电压范围选择**: 24V - 1000V (完整电压范围)
- **极数配置选择**: 2P+E, 3P+E, 3P+N+E, 4P+E, 4P+N+E
- **防护等级选择**: IP44/IP54, IP66/IP67, IP66/IP67/IP69
- **产品类别过滤**: 插头、连接器、暗装插座、明装插座
- **结果展示**: 匹配产品列表，包含技术参数和替换建议

### 3. 反向选型 (ReverseSelectionPage)
**路径**: `/reverse-selection`
**功能**: 基于已知订货号的反向查询和替换
- **订货号输入**: 支持 SCAME 标准订货号格式
- **编码解析**: 自动解析产品技术参数
- **智能替换**: 基于黄金替换法则的推荐
- **技术对比**: 原产品与替换产品的参数对比
- **置信度评分**: 替换建议的可靠度评估

## 核心技术创新

### 1. SCAME 编码规则解析器
**文件**: `src/lib/scame/coding.ts`
**功能**: 完整实现 SCAME 订货号编码规则
- **首位数字映射**: 1-8 分别对应不同产品大类
- **防护等级解析**: 第三位数字 3=IP44/IP54, 8=IP66/IP67/IP69
- **电流规格提取**: 智能匹配 2位/3位电流代码
- **极数配置识别**: 32=2P+E, 36=3P+E, 37=3P+N+E 等
- **产品系列推断**: 基于前缀的系列识别和推断
- **变体标识提取**: 识别订货号末尾的变体标识符

### 2. 智能匹配引擎
**文件**: `src/lib/scame/matching.ts`
**功能**: 产品匹配和替换算法
- **黄金替换法则**: 2↔3↔4↔5 系列互换规则
- **技术参数验证**: 电流、极数、防护等级兼容性检查
- **置信度计算**: 基于参数匹配度的推荐可信度评分
- **差异分析**: 替换产品与原产品的参数差异说明

### 3. RAG 检索增强系统
**文件**: `src/lib/rag/`
**功能**: 基于知识库的智能问答和解释
- **文档处理流水线**: PDF/DOCX/PPTX/TXT → Markdown → 向量化
- **三层检索策略**: 
  1. 结构化产品数据库 (精确匹配)
  2. 向量相似性检索 (语义匹配)
  3. 关键词全文检索 (召回匹配)
- **增强生成**: 基于检索结果的上下文增强回答
- **来源追溯**: 所有回答附带知识来源引用
- **置信度评估**: 回答的准确性和可靠性评分

### 4. 向量存储系统
**文件**: `src/lib/rag/VectorStore.ts`
**功能**: 知识文档的向量化存储和检索
- **开发模式**: MemoryVectorStore (内存存储，快速开发)
- **生产模式**: ChromaDB 接口 (持久化向量数据库)
- **分块策略**: 智能文档分块，保持上下文完整性
- **元数据索引**: 文档来源、类型、时间等元数据管理

### 5. 文档处理器
**文件**: `src/lib/rag/DocumentProcessor.ts`
**功能**: 多格式文档解析和处理
- **格式支持**: PDF, DOCX, PPTX, TXT, Markdown
- **文本提取**: 保持原始格式和结构
- **智能分块**: 基于语义和结构的文档分块
- **质量检查**: 处理结果验证和质量控制

## 使用指南

### 1. 安装和运行

#### 环境要求
- Node.js >= 18.0.0
- pnpm >= 8.0.0 (推荐) 或 npm >= 9.0.0

#### 安装步骤
```bash
# 1. 克隆代码库
git clone <repository-url>
cd scame-selection-tool

# 2. 安装依赖
pnpm install

# 3. 启动开发服务器
pnpm run dev
```

#### 访问地址
- 主地址: http://localhost:3000
- 备用地址: http://192.168.1.11:3000 (根据网络配置)

### 2. 基础选型流程

#### 参数化选型 (正向选型)
1. 导航到 "参数化选型" 页面
2. 选择所需技术参数：
   - 电流规格 (如 32A)
   - 电压范围 (如 200-250V)
   - 极数配置 (如 3P+N+E)
   - 防护等级 (如 IP44/IP54)
   - 产品类别 (如 工业插头)
3. 点击 "开始选型" 按钮
4. 查看匹配产品列表
5. 点击产品查看详情或获取替换建议

#### 反向选型 (基于订货号)
1. 导航到 "反向选型" 页面
2. 输入已知订货号 (如 "213.3237")
3. 系统自动解析技术参数
4. 查看解析结果和替换建议
5. 选择合适的产品进行替换

### 3. RAG 知识库使用

#### 文档上传和管理
1. 导航到 "文档管理" 页面
2. 点击 "上传文档" 按钮
3. 选择 SCAME 官方文档 (PDF/DOCX/PPTX)
4. 等待文档处理完成
5. 查看处理状态和质量报告

#### 知识问答
1. 导航到 "知识库" 页面
2. 在搜索框输入技术问题
3. 查看 RAG 增强的回答
4. 点击来源链接查看原始文档

### 4. 产品对比和选择

#### 多产品对比
1. 在 "产品浏览" 页面勾选多个产品
2. 点击 "对比产品" 按钮
3. 查看技术参数对比表格
4. 基于对比结果做出选择

#### 加入购物车
1. 在产品详情页点击 "加入购物车"
2. 或使用批量选择功能
3. 购物车支持跨页面操作
4. 导出购物车为采购清单

## 编码规则详解

### 1. 订货号结构
```
格式: XXX.YYYYY
示例: 213.3237

分解:
- 前缀 (XXX): 213
  - 首位数字 (2): 输入端/工业插头/器具插座
  - 第二位数字 (1): 系列标识 (1=OPTIMA标准型)
  - 第三位数字 (3): 防护等级 (3=IP44/IP54)

- 后缀 (YYYYY): 3237
  - 前2-3位 (32): 电流规格 (32A)
  - 中间2位 (37): 极数配置 (37=3P+N+E)
  - 最后1位 (可选): 变体标识
```

### 2. 产品类别映射
| 首位数字 | 产品类别               | 说明                     |
|----------|------------------------|--------------------------|
| 1        | 民用面板及附件         | 家用和商业应用           |
| 2        | 输入端/工业插头        | 插头类产品               |
| 3        | 移动输出端/连接器      | 连接器类产品             |
| 4        | 暗装输出端/面板插座    | 暗装插座                 |
| 5        | 明装及高阶控制端       | 明装插座和控制设备       |
| 6        | 箱体与基座             | 外壳和安装底座           |
| 7        | 线缆盘与照明           | 电缆盘和照明设备         |
| 8        | 安装辅材类             | 安装附件和辅材           |
| 899      | 大电流产品             | 特殊大电流系列           |

### 3. 防护等级编码
| 第三位数字 | 防护等级          | 应用场景                 |
|------------|-------------------|--------------------------|
| 3          | IP44/IP54         | 防溅水，室内和遮蔽场所   |
| 8          | IP66/IP67/IP69    | 防水，户外和恶劣环境     |

### 4. 黄金替换法则
```
插头 (2xx) ↔ 连接器 (3xx) ↔ 暗装插座 (4xx) ↔ 明装插座 (5xx)

替换规则:
- 技术参数保持不变 (电流、极数、防护等级)
- 仅改变产品类型和应用场景
- 必须验证系列兼容性
```

## 开发经验总结

### 1. 垂直行业选型工具开发模式

#### 核心洞察
1. **领域知识先行**: 先深入理解行业编码规则和技术规范
2. **约束驱动设计**: 将行业约束转化为系统验证规则
3. **专家协同**: 与领域专家紧密合作，确保技术准确性
4. **数据质量优先**: 高质量的结构化数据是智能选型的基础

#### 开发模式提炼
```typescript
// 模式：约束驱动的选型引擎
interface SelectionEngine<T> {
  // 1. 编码规则解析
  parseCode(code: string): ProductSpec;
  
  // 2. 约束验证
  validateConstraints(spec: ProductSpec): ValidationResult;
  
  // 3. 智能匹配
  findMatches(spec: ProductSpec): T[];
  
  // 4. 替换建议
  suggestReplacements(original: T, context: SelectionContext): Suggestion[];
  
  // 5. 置信度评估
  calculateConfidence(suggestion: Suggestion): number;
}
```

## 故障排除

### 常见问题

#### 1. 页面空白或加载失败
**可能原因**: 
- 开发服务器未启动
- 浏览器缓存问题
- 端口冲突

**解决方案**:
```bash
# 检查服务器状态
pnpm run dev

# 清除浏览器缓存
# 或使用无痕模式访问

# 检查端口占用
lsof -i :3000
```

#### 2. 订货号解析错误
**可能原因**:
- 输入格式不正确
- 电流代码无法识别
- 极数配置无效

**解决方案**:
- 确认订货号格式: `XXX.YYYYY`
- 参考编码规则文档
- 检查电流和极数是否在支持范围内

#### 3. RAG 系统无响应
**可能原因**:
- 文档未上传或未处理
- 向量数据库连接问题
- 检索查询无结果

**解决方案**:
1. 检查文档管理页面处理状态
2. 确认开发环境下使用 MemoryVectorStore
3. 尝试更具体的关键词查询
