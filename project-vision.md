# SCAME 智能选型工具 - 项目愿景

## 一、项目目标

### 短期（0-3 个月）
**工业安全底线**：实现 SCAME 选型100% 精准，零幻觉、零编造型号、零参数错位，杜绝安全事故
**产品闭环**：完成网页版 MVP 上线，正向 / 反向 / 模糊查询全流程可用
**双向赋能**：用 MVP 提升自身 SCAME 销售工作效率，同时验证创业产品可行性

### 中期（3-12 个月）
**创业验证**：打磨出工业级垂直数字员工开发 SOP，完成 SCAME 选型产品的商业闭环
**能力沉淀**：具备快速复制能力，可无缝适配施耐德等其他电气元器件品牌

### 长期（1-3 年）
**商业扩张**：以 BaaS 模式向小微电气代理商 / 五金商户规模化交付数字员工
**行业地位**：成为工业元器件垂直领域AI 数字员工劳务派遣标杆

- 五金商贸城 5-10 人小微公司 / 门店（无技术能力、无 AI 学习成本、刚需高效选型）

## 三、成功标准（工业级 + 商业级 + 职场级三维考核）

### 1. 工业安全核心标准（一票否决）
- 选型精准率 100%，正向 / 反向 / 模糊查询无任何型号 / 参数错误，零安全事故风险
- 彻底杜绝 LLM 幻觉、编造型号、串用参数等致命问题

### 2. 产品可用标准
- 网页版稳定运行，全流程操作零障碍
- 库存 / 报价模块无缝对接，数据保密、更新便捷
- 多渠道接入（优先钉钉 / 飞书）可快速落地

### 3. 创业验证标准
- 形成可复制的工业选型数字员工开发模板
- 可无缝迁移至其他电气元器件品牌，无需从零开发

### 4. 职场赋能标准
- 替代 80% 以上手动选型 / 报价工作，个人销售效率提升 10 倍
- 全面吃透 SCAME 全产品体系，强化职场核心竞争力

## 四、技术约束和优先事项

### 技术约束（刚性不可突破）
- **安全约束**：电气选型100% 精准，禁止任何可能引发事故的参数错误
- **数据约束**：库存 / 报价等敏感数据云端加密存储，保密、易更新、多租户隔离
- **合规约束**：符合工业电气产品选型规范，严格遵循 SCAME 产品编码规则
- **轻量化约束**：适配小微用户，零安装、零培训、即开即用，无复杂部署
- **成本约束**：轻资产开发，无大额算力 / 服务器成本，适配一人创业模式

### 技术优先事项（CTO 级排期，按生死优先级排序）
1. **第一优先级（生死线）**：结构化 SCAME 12000 行产品数据，用规则引擎 + 数据库实现 100% 精准选型，彻底解决串号 / 幻觉问题
2. **第二优先级**：打通选型→参数→库存的核心链路，保证基础功能闭环
3. **第三优先级**：开发智能报价、报价单导出功能，完成商业闭环
4. **第四优先级**：UI 美化、网页上线、服务器稳定部署
5. **第五优先级**：多渠道机器人接入（钉钉 / 飞书）、产品技术单页完善
6. **第六优先级**：沉淀标准化 SOP，为跨品牌复制做准备
