# SCAME 智能选型工具

## What This Is

SCAME 智能选型工具是为工业电气元器件行业开发的100%精准选型系统，通过规则引擎和结构化产品数据库实现零幻觉、零误差的工业安全级选型。服务于 SCAME 销售团队、渠道代理商和小微电气商户，提供正向选型、反向选型、模糊查询全流程功能。

## Core Value

**工业安全底线优先**：电气选型必须100%精准，零幻觉、零编造型号、零参数错位，彻底杜绝安全事故风险。

**工业约束**：

## Constraints

- **安全约束**：电气选型100%精准，禁止任何可能引发事故的参数错误
- **数据约束**：库存/报价等敏感数据云端加密存储，保密、易更新、多租户隔离
- **合规约束**：符合工业电气产品选型规范，严格遵循SCAME产品编码规则
- **轻量化约束**：适配小微用户，零安装、零培训、即开即用，无复杂部署
- **成本约束**：轻资产开发，无大额算力/服务器成本，适配一人创业模式
